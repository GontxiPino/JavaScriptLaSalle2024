Apuntes de JavaScript Strings

Definición

String: Una secuencia de caracteres usados para representar texto.
Sintaxis: Se pueden definir usando comillas simples (' ') o dobles (" "), y a partir de ES6, también con backticks (` `) para templates literals.

Propiedades

  • length: Retorna la longitud del string. 
    let str = "Hello, World!";
    console.log(str.length); // 13

Métodos Principales

  • charAt(index): Devuelve el carácter en el índice especificado. 
    console.log(str.charAt(0)); // "H"
  • charCodeAt(index): Devuelve el valor Unicode del carácter en el índice especificado. 
    console.log(str.charCodeAt(0)); // 72
  • concat(str1, str2, ...): Combina dos o más cadenas de texto y devuelve una nueva cadena. 
    let str1 = "Hello";
    let str2 = "World";
    console.log(str1.concat(", ", str2)); // "Hello, World"
  • includes(substring, start): Comprueba si una cadena contiene otra cadena. 
    console.log(str.includes("World")); // true
  • indexOf(substring, start): Devuelve el índice de la primera aparición de una subcadena. 
    console.log(str.indexOf("World")); // 7
  • lastIndexOf(substring, start): Devuelve el índice de la última aparición de una subcadena. 
    console.log(str.lastIndexOf("o")); // 8
  • match(regex): Devuelve un array con las coincidencias encontradas en la cadena. 
    console.log(str.match(/l/g)); // ["l", "l", "l"]
  • replace(searchValue, newValue): Reemplaza coincidencias de una cadena con otra cadena. 
    console.log(str.replace("World", "Universe")); // "Hello, Universe!"
  • search(regex): Busca una coincidencia entre una expresión regular y la cadena. 
    console.log(str.search("World")); // 7
  • slice(start, end): Extrae una sección de una cadena y devuelve una nueva cadena. 
    console.log(str.slice(0, 5)); // "Hello"
  • split(separator, limit): Divide una cadena en un array de subcadenas. 
    console.log(str.split(" ")); // ["Hello,", "World!"]
  • substring(start, end): Devuelve una parte de la cadena entre los índices especificados. 
    console.log(str.substring(0, 5)); // "Hello"
  • toLowerCase(): Convierte toda la cadena a minúsculas. 
    console.log(str.toLowerCase()); // "hello, world!"
  • toUpperCase(): Convierte toda la cadena a mayúsculas. 
    console.log(str.toUpperCase()); // "HELLO, WORLD!"
  • trim(): Elimina los espacios en blanco de ambos extremos de una cadena. 
    let strWithSpaces = "   Hello, World!   ";
    console.log(strWithSpaces.trim()); // "Hello, World!"

Ejemplo Completo

let greeting = " Hello, JavaScript! ";

console.log(greeting.length); // 20
console.log(greeting.charAt(1)); // "H"
console.log(greeting.includes("JavaScript")); // true
console.log(greeting.indexOf("JavaScript")); // 8
console.log(greeting.replace("JavaScript", "World")); // " Hello, World! "
console.log(greeting.slice(1, 6)); // "Hello"
console.log(greeting.split(" ")); // ["", "Hello,", "JavaScript!"]
console.log(greeting.toLowerCase()); // " hello, javascript! "
console.log(greeting.toUpperCase()); // " HELLO, JAVASCRIPT! "
console.log(greeting.trim()); // "Hello, JavaScript!"
