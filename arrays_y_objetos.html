Apuntes de Arrays y Objetos en JavaScript

Arrays

Definición

Un array es una colección de elementos ordenados que pueden ser de cualquier tipo de datos.

Declaración y Creación

  • Declaración de un array: 
    let frutas = ["Manzana", "Banana", "Cereza"];
  • Declaración de un array vacío: 
    let numeros = [];

Acceso a Elementos

Se accede a los elementos de un array mediante su índice (comenzando desde 0). 

let primeraFruta = frutas[0]; // "Manzana"

Propiedades y Métodos Comunes

  • length: Retorna el número de elementos en el array. 
    console.log(frutas.length); // 3
  • push(elemento): Agrega un elemento al final del array. 
    frutas.push("Durazno"); // ["Manzana", "Banana", "Cereza", "Durazno"]
  • pop(): Elimina el último elemento del array. 
    let ultimaFruta = frutas.pop(); // "Durazno"
  • shift(): Elimina el primer elemento del array. 
    let primeraFruta = frutas.shift(); // "Manzana"
  • unshift(elemento): Agrega un elemento al inicio del array. 
    frutas.unshift("Fresa"); // ["Fresa", "Banana", "Cereza"]
  • forEach(función): Ejecuta una función para cada elemento del array. 
    frutas.forEach(function(fruta) {
        console.log(fruta);
    });
  • map(función): Crea un nuevo array con los resultados de la función aplicada a cada elemento. 
    let frutasEnMayusculas = frutas.map(function(fruta) {
        return fruta.toUpperCase();
    }); // ["FRESA", "BANANA", "CEREZA"]

Objetos

Definición

Un objeto es una colección de pares clave-valor. Cada clave debe ser única dentro del objeto.

Declaración y Creación

  • Declaración de un objeto: 
    let persona = {
        nombre: "Juan",
        edad: 30,
        profesion: "Desarrollador"
    };

Acceso a Propiedades

Se accede a las propiedades de un objeto mediante la notación de punto o la notación de corchetes. 

console.log(persona.nombre); // "Juan"
console.log(persona["edad"]); // 30

Adición y Modificación de Propiedades

  • Adición de una nueva propiedad: 
    persona.nacionalidad = "Española";
  • Modificación de una propiedad existente: 
    persona.edad = 31;

Eliminación de Propiedades

delete persona.profesion;

Métodos de Objetos

Los objetos pueden contener métodos, que son funciones asociadas a un objeto. 

let persona = {
    nombre: "Juan",
    edad: 30,
    saludar: function() {
        console.log("Hola, soy " + this.nombre);
    }
};
persona.saludar(); // "Hola, soy Juan"

Iteración sobre Propiedades

Se puede iterar sobre las propiedades de un objeto utilizando un bucle for...in. 

for (let clave in persona) {
    console.log(clave + ": " + persona[clave]);
}

Ejemplo Completo

// Arrays
let frutas = ["Manzana", "Banana", "Cereza"];
frutas.push("Durazno");
frutas.forEach(function(fruta) {
    console.log(fruta);
});

// Objetos
let persona = {
    nombre: "Juan",
    edad: 30,
    profesion: "Desarrollador",
    saludar: function() {
        console.log("Hola, soy " + this.nombre);
    }
};
persona.saludar();
for (let clave in persona) {
    console.log(clave + ": " + persona[clave]);
}
